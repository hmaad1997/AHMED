# Quran Reciter ID - Flutter App

## 📱 تطبيق Android للتعرف على صوت القارئ

هذا هو التطبيق الأمامي (Frontend) المبني بـ Flutter للتواصل مع Backend Server.

---

## 🚀 خطوات التشغيل

### 1. تثبيت المكتبات

```bash
cd flutter_app
flutter pub get
```

**الوقت المتوقع:** 1-2 دقيقة

---

### 2. تحديث عنوان الـ Backend

افتح ملف: `lib/services/api_service.dart`

**للـ Android Emulator:**
```dart
static const String baseUrl = 'http://10.0.2.2:8000';
```

**لجهاز حقيقي (Physical Device):**
```dart
static const String baseUrl = 'http://192.168.x.x:8000';
// استبدل 192.168.x.x بعنوان IP الخاص بجهاز الكمبيوتر
```

**للاختبار على الكمبيوتر:**
```dart
static const String baseUrl = 'http://localhost:8000';
```

---

### 3. تشغيل Backend Server

**قبل تشغيل التطبيق، يجب تشغيل Backend Server:**

```bash
# في terminal منفصل
cd backend
python run_server.py
```

**تأكد من الرسالة:**
```
✓ SERVER READY
Uvicorn running on http://0.0.0.0:8000
```

---

### 4. تشغيل التطبيق

**على Android Emulator:**
```bash
flutter run
```

**على جهاز حقيقي:**
1. صل الجهاز بـ USB
2. فعّل "USB Debugging"
3. شغّل:
```bash
flutter run
```

**الوقت المتوقع لأول تشغيل:** 3-5 دقائق (بناء APK)

## 🎨 ميزات التطبيق

### الشاشة الرئيسية (Home)
- ✅ تبديل بين التسجيل والمكتبة
- ✅ مؤشر حالة الاتصال بالسيرفر (أخضر/أحمر)

### شاشة التسجيل (Recording)
- ✅ تسجيل صوتي مباشر
- ✅ عرض مؤقت التسجيل
- ✅ تأثيرات بصرية (Pulse animation)
- ✅ التحقق من الحد الأدنى للتسجيل (3 ثواني)
- ✅ إرسال الصوت للـ Backend

### شاشة النتيجة (Result)
- ✅ عرض اسم القارئ (عربي + إنجليزي)
- ✅ نسبة التطابق (Confidence %)
- ✅ معلومات القارئ:
  - البلد
  - سنوات الميلاد/الوفاة
  - نمط التلاوة
  - نبذة تعريفية

### شاشة المكتبة (Library)
- ✅ قائمة بجميع الـ 50 قارئ
- ✅ بحث بالاسم أو البلد
- ✅ عرض تفاصيل كل قارئ
- ✅ سحب للتحديث (Pull to Refresh)

## 🧪 اختبار التطبيق

### اختبار 1: حالة السيرفر
1. شغّل التطبيق بدون Backend
2. يجب أن ترى نقطة حمراء في الـ AppBar
3. رسالة تحذير: "الخادم غير متصل"

### اختبار 2: التسجيل
1. اضغط على زر الميكروفون
2. اسمح بصلاحيات الميكروفون
3. سجّل 5-10 ثواني من التلاوة
4. اضغط "إيقاف وتحليل"
5. يجب أن ترى شاشة النتيجة

### اختبار 3: المكتبة
1. اذهب لتبويب "مكتبة القراء"
2. يجب أن ترى قائمة بالـ 50 قارئ
3. جرّب البحث
4. اضغط على قارئ لرؤية التفاصيل

## ⚠️ حل المشاكل الشائعة

### ❌ خطأ: "الخادم غير متصل"
**الحل:**
1. تأكد من تشغيل Backend: `python run_server.py`
2. تحقق من عنوان الـ API في `api_service.dart`
3. إذا كنت على جهاز حقيقي، استخدم IP الكمبيوتر

### ❌ خطأ: "Microphone permission denied"
**الحل:**
1. اذهب لإعدادات التطبيق
2. فعّل صلاحية الميكروفون

### ❌ خطأ: "Audio too short"
**الحل:**
- سجّل 3 ثواني على الأقل

### ❌ خطأ: Flutter not found
**الحل:**
```bash
# تثبيت Flutter
# https://docs.flutter.dev/get-started/install
```

### ❌ التطبيق بطيء
**الحل:**
- Backend يحتاج 2-5 ثواني لتحليل الصوت (هذا طبيعي)
- تأكد من اتصال إنترنت/شبكة جيد
